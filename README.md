## Description

Unit and Integration Testing with Nest.js and Vitest

## Installation

```bash
$ yarn install
```

## Running the app

```bash
$ yarn run start
```

## Run Unit Test

```bash
$ yarn run test
```

## Run Integration Test

```bash
$ yarn run test:e2e
```

## Tools Used

- Nest.js: Node.js framework for building web applications and web services.
- Vitest: Tool for writing and running tests.
